
# Projeto de Votação de Filmes e Séries

Este é um projeto simples feito em Flask para cadastrar, listar e votar em filmes usando SQLite3 e Python.

---

## Funcionalidades

- Listar filmes cadastrados  
- Adicionar filmes novos com título, gênero, descrição e imagem  
- Votar em filmes com "gostei" ou "não gostei"
